# 第三章:湖夜告白

> 你隐秘的耳语和呼吸，被我系上这一颗心，沉入瓶底。目的是，造一罐，不为后人理解的蜜

---

同班同学的身份让线上那句“恭喜夺冠”后的闲聊，变得顺理成章且日益频繁。话题从令人头疼的C++作业，慢慢延伸到食堂哪个窗口的菜量最足，再到吐槽某个选修课老师的口音。对话框里渐渐充满了生活的细碎片段。

一个周五的晚上，鲤依佳的消息跳了出来：“听说中心广场那边的空地，晚上好多人玩滑板，星星灯很漂亮，想去看看吗？”

月光如练，洒在中心湖畔的空地上。各色LED灯带缠绕在栏杆上，映着一个个飞驰而过的滑板少年身影，确实热闹。鲤依佳眼睛亮晶晶的，从朋友那儿借来一块入门双翘板，跃跃欲试地放在脚边。

“我完全不会哦，你可要看着我点。”她说着，深吸一口气，尝试站上去，板子立刻不听话地往前溜，她惊呼一声，身体失衡，下意识地就抓住了旁边陈葱的胳膊。
“你扶我一下呀，阿葱！”她抬起头，眼神里带着全然的信任和一点不自觉的撒娇。

陈葱只能硬着头皮上前，小心翼翼地扶住她的胳膊。女孩的身体比他想象的还要轻盈，带着刚洗过澡后淡淡的茉莉花洗发水香气。她每次试图找到平衡时，手指都会不自觉地收紧，隔着薄薄的衣料传来温热的触感。她的笑声和偶尔因为差点摔倒发出的小小惊呼，在混杂着轮子摩擦声的晚风里，显得格外清晰，一下下敲在陈葱的心上。

而他这具身体，似乎对一切运动项目都有着天然的亲和力。在扶了她几圈后，他忍不住自己也试了试。虽然动作生涩，但他居然能摇摇晃晃地站在板上滑行一小段距离，没有摔倒。
“哇，你学得好快！”鲤依佳扶着栏杆，语气里是毫不掩饰的羡慕和赞叹，“是不是打篮球的人平衡感都这么好啊？”

他只能含糊地应了一声，内心再次被这具身体的本能所震撼。

如果说滑板夜是带着暧昧试探的意外之旅，那么每周的篮球体育课，则渐渐成了他们心照不宣的固定“约会”。她打球的样子很认真，防守时微微蹙眉，眼神专注；当她投进一个球（尤其是在他助攻下），会立刻转过头，眉眼弯弯地朝他露出一个灿烂的笑容，并开心地与他击掌。
休息间隙，她会很自然地走到场边，拿起他喝了一半的矿泉水，仰头就喝。有次臣菌玉也想拿水，被她眼疾手快地拍开：“这是陈葱的！”动作流畅自然，仿佛天经地义。而当陈葱跟班里其他男生打半场切磋时，她从不掺和，只是安静地坐在场边的树荫下，手里拿着水，眼睛里仿佛只有他一个人奔跑、跳跃、投篮的身影。

一切都温暖而美好，朝着那个似乎早已注定的轨道滑去。直到那个看似平凡的、阳光有些慵懒的下午。

“在宿舍吗？下楼一下，有东西给你。”鲤依佳的消息突兀地弹出，后面跟了一个小猫探头的表情包。

陈葱的心跳莫名漏了一拍，某种预感攫住了他。他快步下楼，一眼就看到站在宿舍楼门口榕树下的女孩。她背着手，脚尖无意识地碾着一颗小石子，看到他出现，脸上立刻绽开一个比午后阳光还明媚几分的笑靥。

“喏，奖励你上次拍摄辛苦啦。”她将背在身后的手拿出，递过来一杯冰凉的葡萄芋圆冻冻，杯壁上凝结着诱人的水珠。

他刚接过奶茶，指尖碰到她微凉的皮肤，一句“谢谢”还没说出口，就听到她声音不大，却异常清晰地说道：

“陈葱，我喜欢你。”

“……”

世界仿佛被按下了静音键。周围同学的说笑声、车辆的喇叭声瞬间远去。陈葱彻底懵了，大脑像被格式化的硬盘，一片空白。理智在深处微弱地发出警报，提醒着他所知的“历史”中那段无疾而终的初恋。拒绝吗？以什么理由？

然而，他的身体却抢先一步，背叛了所有混乱的思绪。一股灼热感不受控制地从心脏炸开，汹涌地冲向四肢百骸，脸颊和耳根迅速发烫，他甚至能感觉到自己脖颈后的血管在突突跳动。这具身体，正用最原始、最直白不过的生理反应，表达着它的狂喜与全盘接纳。这是属于陈葱的本能，对眼前这个女孩最直接的共鸣。

“我……你……你先让我想想！”他几乎是语无伦次，甚至不敢看她的眼睛，攥着那杯冰凉的奶茶，像逃跑一样转身冲回了宿舍楼。

他需要援军。他找到正在宿舍里研究穿搭的贺朋非。贺朋非听完他磕磕巴巴的叙述，难得地收起了平时那副玩世不恭的神色，拍了拍他的肩膀：“兄弟，人家女生都这么勇敢了，你不能怂。无论答应与否，都要堂堂正正跟人家说清楚。表白这种事，本该是男生来的，咱得有点担当。”

傍晚，他和臣菌玉坐在食堂角落吃着麻辣香锅，依旧心乱如麻，食不知味。臣菌玉话不多，只是默默把香锅里的鱼豆腐都夹到他碗里，说：“跟着感觉走，葱哥。”

吃完饭，两人慢悠悠地走回宿舍。夕阳将天空染成暖橙色，校道旁，有小贩正在售卖为第二天六一儿童节准备的彩色氢气球，一簇簇鲜艳的色彩漂浮在半空，随着晚风轻轻摇曳。看着那些无忧无虑、象征着最纯粹快乐的气球，看着路过的小孩子牵着气球时脸上天真烂漫的笑容，一股莫名的、近乎幼稚的勇气，突然从他心底升腾起来。

去他的历史！去他的结局！眼前的心动和真诚，才是真实的！
就是她了。

这个念头一旦升起，便再也无法压制。

那天晚上，他跑到学校后门的花店，买了一束不算很大、但每一朵都新鲜的玫瑰。他叫上了贺朋非和臣菌玉——一个负责用手机全程记录这历史性的一刻，一个则作为后勤部长，在关键时刻从背后为他递上花束。

还是那个校道湖畔，还是那排轻柔拂动的柳树下，晚风比滑板那夜更加温柔。

鲤依佳刚下选修课，脸上还带着些许期待。当她看到站在柳树下、手里背在身后、神情紧张却目光坚定的陈葱时，似乎明白了什么，脸颊瞬间飞上红霞。

贺朋非的镜头已经就位，臣菌玉在他身后比了个大拇指。

陈葱深吸一口气，走上前，将藏在身后的那束玫瑰花拿到身前，递向她。

“鲤依佳，”他的声音因为紧张而有些微哑，但每一个字都说得无比清晰、坚定，“之前……是你说的。现在，换我来说。”
他停顿了一下，凝视着她因为惊讶而微微睁大的眼睛，继续说道：
“我喜欢你。做我女朋友，好吗？”

那一刻，驱动他开口、让他鼓起所有勇气的，是这具身体里奔涌的、无法抑制的原始爱意。

去他妈的莱奇那个灵魂！

身体仿佛有自己的意识，它笨拙却无比真诚的、想要抓住这份美好。

鲤依佳看着他，眼里的惊讶慢慢化为几乎要溢出来的甜蜜笑意，她用力地点了点头，接过花束，声音轻快而肯定：“好！”

贺朋非在一旁激动地差点喊出来，赶紧稳住镜头。臣菌玉也露出了憨厚的笑容。

当晚，贺朋非就爆手速将表白视频剪成了一个节奏明快、氛围感拉满的恋爱Vlog，配上学姐推荐的、氛围感绝佳的宇多田光《One Last Kiss》，在兄弟几个的小群里疯狂传播，收获了满屏的“99”。

正式脱单后，陈葱的大学生活模式彻底更新。上课的固定座位从后排的“兄弟连”变成了前排的“情侣座”。他再也没能和臣菌玉、贺朋非他们在课堂上用手机开黑或者插科打诨，引得两位舍友在宿舍群里连连“诟病”他“重色轻友”、“有了媳妇忘了爹”。

而在学业上，尤其是在编程领域，另一种“本能”也开始以前所未有的势头觉醒。他像一匹横空出世的黑马，被那位爱穿拖鞋、指尖常夹着烟的老师一眼相中，直接引入了名为“珠江学者培养创新计划”的学霸聚集地。大一下学期，属于“陈葱”的传奇开始全面绽放，他几乎以横扫之势，接连拿下了蓝桥杯全国软件和信息技术专业人才大赛国赛一等奖、全国大学生计算机程序设计大赛金奖、ACM国际大学生程序设计竞赛区域赛冠军以及全国大学生数学建模竞赛一等奖等多个极具分量的奖项，成为了整个广东医大计算机相关领域无人不知的新星。

期末的编程基础考试，那位拖鞋烟枪老师出了一道让所有学生都会心一笑的浪漫题目：在线写出一个爱心代码，不论语言与形式。

考场里瞬间响起一片敲击键盘的声音，夹杂着些许兴奋的低语。

鲤依佳在陈葱平日里的耐心熏陶和“开小灶”下，编程水平已然大大提高。她凝神静气，回想著书本上的循环和条件判断知识，用C++写下了一段简洁而真诚的代码：

```cpp
#include <iostream>
#include <cmath>

int main() {
    const int scale = 6;
    for (double y = 1.5; y > -1.5; y -= 0.1) {
        for (double x = -1.5; x < 1.5; x += 0.05) {
            double a = x * x + y * y - 1;
            if (a * a * a - x * x * y * y * y <= 0)
                std::cout << '*';
            else
                std::cout << ' ';
        }
        std::cout << std::endl;
    }
    return 0;
}
```

运行后，屏幕上会显示一个由星号构成的、朴素的爱心图案：

```


                *********           *********
            *****************   *****************
          ****************************************
         *******************************************
        *********************************************
        *********************************************
        *********************************************
         *******************************************
          *****************************************
            *************************************
             ***********************************
                *****************************
                  *************************
                    *********************
                       ***************
                          *********
                             ***


```

虽然简单，却充满了她的用心。她满意地提交了代码，感觉这次考试格外顺利。

而坐在不远处的陈葱，甚至不需要过多思考，手指仿佛自有主张，在键盘上行云流水般敲击，逻辑严谨、结构优美的代码迅速成型，他流畅地提交，获得了毫无悬念的满分。

考完试，鲤依佳心情雀跃，跑到陈葱身边，好奇地想看他写的爱心是什么样子。“让我看看你的嘛，肯定比我的复杂多了！”

陈葱刚想点头，却被不知从哪儿冒出来的臣菌玉一把搂住脖子：“葱哥，江湖救急！球场三缺一，就差你这个王牌了！嫂子，借你家陈葱用一下哈！”说着就要把人拖走。
陈葱无奈，只好赶紧把保存在手机里的代码文件发到鲤依佳的微信上，“你自己先看，我去去就回！”

女生宿舍里，鲤依佳和她的室友们迫不及待地围在电脑前，好奇地点开了那个文件。

代码确实比她的要长一些，结构也更复杂，嵌套了多个循环来控制字符的疏密和形状的饱满度。一个室友帮忙点击了编译运行。

黑色的命令行窗口弹出，短暂的加载后，一个由粒子效果渲染而成的爱心，稳稳地呈现出来。它跳动着，那明暗的动态效果，仿佛蕴含着某种沉静而强大的力量，一种独属于陈葱的、内敛而又激情的表达。

看着屏幕上那颗为他而存在的、独一无二的“心”，再对比自己代码输出的那个略显青涩的爱心，鲤依佳清晰地感觉到，自己的心脏正随着代码的爱心砰砰作响，如同怀里揣了一只不安分的小鹿，撞击着她的胸腔。

那是代码最原始的浪漫，是独属于他的、在冷静逻辑下包裹的温柔内核，也是她在这个初夏的期末时节，收获的最硬核、最让她心动的甜蜜答案。她知道，这颗动态的爱心，比任何礼物，都更符合陈葱的风格，也更能触动她的心弦。
